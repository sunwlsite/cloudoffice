# cloudoffice

> element-ui版本: 2.15.1
>
> axios版本: 0.21.1
>
> 其它依赖的版本请查看: package.json

### 依赖安装
```
npm install
```

### 编译运行(热加载)
```
npm run serve
```

### 编译构建
```
npm run build
```
